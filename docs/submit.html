Submit a model

One submission is 9 files — the 3 fluxes (GPP, ET, NEE) for each of the three scenarios (temporal, spatial, temperature). These are the raw predictions you get by running the benchmark on your model — see the GitHub repo. We require a link to your code so we can check there is no test-set leakage (this check is done after your scores are posted), and an email in case of questions or issues. The leaderboard should reflect your scores in less than 30 minutes.
