FLUXNET ML Leaderboard

This leaderboard tracks machine-learning model performance on the FLUXNET benchmark across three evaluation settings (temporal split, spatial split, temperature-based split) and three carbon/water flux targets (GPP, ET, NEE). Metrics are pre-computed by submitters and validated automatically on pull request. See the GitHub repository for submission instructions.
